Notes from Basics of the Web and HTML

Computers are stupid
A computer can only do exactly what you tell it to. If you're not getting the results you want either your syntax is incorrect or you don't understand the rules that govern the computer's response.

Making the Web Work
The web is an interaction of computers. It starts when a computer sends a request for a specific documnent. This request is sent via HTTP to a server. The server locates the page and sends it back to the computer. The computer and it's browser then translate the page's code and render the images, text, and layout accordingly.

HTML Markup: Tags & Elements

Tags and elements are used to organize the content of the document. Tags usually include an opening tag and a closing tag.
The two tags combined are often referred to as an "element".

Attributes
Some tags utilize attributes.
Anchors are examples of tags that use attributes. These attributes can determine whether a link opens in the current window, a new window etc. This link to Google should open in a new browser tab.
Images are also examples of tags that use attributes. Source, hidth & width can all be attributes. This photo is substantially larger than is shown, but has been scaled down using attributes

The img tag is one example of a tag that only has an opening tag

Block Vs. Inline
Block elements establish and invisible "box" around their contents. This can provide a more obvious seperation within the body of a documnent. The BOLDED headers and their subsequent paragraphs are examples of the p tag, a type of Block tag.
This line is seperated from the previous line with a br tag; effectively a 'line break' that maintains the flow of the paragraph created with the p tag.
